\doxysubsection{Detailed Description}
Hybrid string implementation that can be used in both compile time and runtime. 

This header defines a {\bfseries{hybrid constexpr string}} type that supports operations in both compile-\/time and runtime environments. It is primarily designed for {\bfseries{constexpr metaprogramming}} and {\bfseries{string literal manipulation}}, so runtime functionality will be limited.

\begin{DoxyNote}{Note}
Requires {\bfseries{C++17 or later}} -\/ C++20 is recommended, with C++17 is supported with limited constexpr metaprogramming
\end{DoxyNote}
